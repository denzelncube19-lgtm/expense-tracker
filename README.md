# Expense Tracker v1

My first Python project, built as a first-year CSE student.

## Features
- Add expenses with name, amount, and date
- View all expenses with total, highest, and lowest spending
- View today's expenses with a daily total
- Auto-saves to a file so data persists between sessions

## How to run
1. Make sure Python is installed
2. Download `expense_tracker.py`
3. Run in terminal: `python expense_tracker.py`

## What I learned
- Functions and program structure
- File I/O (reading and writing to .txt files)
- Input validation and error handling
- Working with lists and dictionaries
- Date handling with the `time` module
